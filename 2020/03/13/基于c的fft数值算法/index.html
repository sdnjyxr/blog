基于C++的FFT数值算法

     二十世纪中叶以来，信号处理快速算法研究取得长足进展，其中包括DFT的快速算法，即快速Fourier变换（Fast Fourier Transform， FFT），FFT是各种DFT快速算法的统称。

    N点序列x[k]的DFT与IDFT分别定义为：

$$X[m] = \sum\limits_{k=0}^{N-1} x[k]W_{N}^{km}, m = 0, 1,\cdots, N - 1 \tag{1} $$

$$x[k] = \frac{1}{N}\sum\limits_{k=0}^{N-1} X[m]W_{N}^{-km}, k = 0, 1,\cdots, N - 1 \tag{2} $$

基2时间抽取FFT算法原理

    基2时间抽取（Decimation In Time, DIT）的算法原理是在时域将序列逐次分解为两个子序列，利用旋转因子( W_{N}^{mk} ) 的特性，由子序列的DFT来逐次合成实现整个序列的DFT，从而提高DFT的运算效率。旋转因子(W_{N}^{mk} = e^{-j\frac{2\pi}{N}mk})具有周期性、对称性及可约性等特性。

（1）旋转因子 ( W_{N}^{mk} ) 以N为周期，即

[ W_{N}^{mk} = W_{N}^{k(N+m)} = W_{N}^{m(N+k)} ]

（2）旋转因子 ( W_{N}^{mk} ) 存在对称性，即

[ W_{N}^{mk+\frac{N}{2}} = - W_{N}^{mk} , \quad ( W_{N}^{mk} )^{*} = W_{N}^{-mk} ]

（3）旋转因子 ( W_{N}^{mk} ) 具有可约性，即

[ W_{N}^{mk} = W_{nN}^{nmk}, \quad W_{N}^{mk} = W_{N/n}^{mk/n} \quad (N/n \ is \ Integer) ]

    基于以上特性可得

[ X[m] = X_{1}[m] + W_{N}^{m} \cdot X_{2}[m], \quad X[m+N/2] = X_{1}[m] - W_{N}^{m} \cdot X_{2}[m], \quad m = 0, 1, \cdots, N/2 - 1 ]

     当表示N=8点序列x[k]的分解过程时，可将序列x[k]的序号用3位二进制数表示，即x[000]、x[001]、…… 、x[111]。若已知某数的倒序数是J，求下一个倒序数，应先判断J的最高位是否为0，与k=N/2进行比较即可得到结果。如果k>J，说明最高位为0，应把其变成1，即J+N/2，这样就得到倒序数了。如果k<=J，即J的最高位为1，将最高位化为0，即J-N/2，再判断次高位；与k=N/4进行比较，若为0，将其变位1，即J+N/4，即得到倒序数，如果次高位为1，将其化为0，再判断下一位……

第二步 按运算流图运算

     根据旋转因子的特性，图1中的( W_{8}^{0} )可以替换成 ( W_{2}^{0} ) ，( W_{8}^{2} )可以替换成 ( W_{4}^{1} ) 。由此可以推广到对任意(2^{n})个数据，都可以在第一级写做 ( W_{2}^{0} ) ，在第二级写做 ( W_{4}^{0} ) 和 ( W_{4}^{1} ) ，在第三级写做 ( W_{8}^{0} ) 、( W_{8}^{1} ) 、 ( W_{8}^{2} ) 、 ( W_{8}^{3} ) ，以此类推。

     对于N点的FFT需要(log_{2}^{N})级运算。基于以上理论，第一级转换因子都为 ( W_{2}^{0} ) ，第二级转换因子分别为 ( W_{4}^{0} ) 和 ( W_{4}^{0} W_{4}^{1} ) ，第三级转换因子分别为 ( W_{8}^{0} ) 、 ( W_{8}^{0} ) ( W_{8}^{1} ) 、( W_{8}^{0} ) ( W_{8}^{1} ) ( W_{8}^{1} ) 、 ( W_{8}^{0} ) ( W_{8}^{1} ) ( W_{8}^{1} ) ( W_{8}^{1} ) 。

     根据欧拉公式( e^{-\frac{2\pi}{mmax}}=e^{j\theta}=cos\theta+jsin\theta )，即可根据复数的乘法运算法则进行三角迭代运算。
